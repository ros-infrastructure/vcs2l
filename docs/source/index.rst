vcs2l
=====

vcs2l provides a CLI tool to invoke vcs (version control system) commands on multiple repositories.
It currently supports `Git <http://git-scm.com>`_, `Mercurial <https://www.mercurial-scm.org>`_,
`Subversion <http://subversion.apache.org>`_, and `Breezy <https://www.breezy-vcs.org>`_.
